曾在亚马逊云科技（AWS）上海 Summit 及User Group活动中担任讲师，积极推动技术传播与开发者社区建设。

日常斜杠：写文章拿过奖，修过电脑，家电，写过代码，玩过硬件，3D 设计，做过课程讲师，公开演讲，上架过小程序和微信表情包。
